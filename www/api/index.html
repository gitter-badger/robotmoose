RobotMoose API (Application Programming Interface)

States

Your code runs in one state at a time.  It keeps running the state until the state times out, or you switch to a new state.


state = turnLeft

  • runs the state named turnLeft


stop()
  • stops the code, exiting your states


Basic User Interface (UI)

button("string", turnLeft)
  • string is button's title, pressing the button will run the state named turnLeft


checkbox("string")
  • string is the checkbox's title


label("string")
  • string will appear in the UI window


print("string")
  • string will appear to the right side of the state in the code editor


slider("string", -1.0, 0.0, 1.0)
  • creates a slider in the UI titled string, with range -1.0 to 1.0 and default slider placement at 0.0


store.x=j
  • stores a variable named x, initialized to value j


Create2

forward(10)  // drive forward 10 centimeters
backward(5)  // drive backward 5 centimeters
left(45)  // turn left 45 degrees
right(90)  // turn right 90 degrees

drive(-0.5, 1.0)  // sets the speed of the left wheel to -0.5, sets the speed of the right wheel to +1.0 (range is -1.0 to 1.0)


PWM (Pulse Width Modulation)

power.pwm[i] = j
  • sets the value of pwm[i] to j (range is 0 to 255)


Servos

power.servo[i] = j
  • moves servo[i] to position j (in degrees, range is 0 to 180)


Control Flow

if(expression)
{
    statement1
}
else
{
    statement2
}
  • if expression evaluates to true, executes statement1, otherwise executes statement2
